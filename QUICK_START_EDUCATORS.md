# Quick Start Guide for Educators

## Creating Your First Course with Modules, Lessons, Quizzes & Assessments

### Step 1: Navigate to Upload Courses
1. Login to your educator dashboard
2. Click on "Home" in the sidebar
3. Select "Upload courses" from the top menu

### Step 2: Fill Course Details (Step 1 of 3)
Fill in the basic information:
- **Course Title**: Give your course a clear, descriptive name
- **What You'll Learn**: List key learning outcomes (bullet points work great)
- **Ideal For**: Describe your target audience
- **Skill Level**: Choose from Beginner, Intermediate, Advanced, or All Levels
- **Tools Used**: List any tools or software students will need

Click **Next** to continue.

### Step 3: Build Course Structure (Step 2 of 3)

#### Add Course Description
- Write a comprehensive description of what the course covers
- Explain what's included (videos, quizzes, certificates, etc.)

#### Create Modules
Click **"Add Module"** to open the module builder.

##### Module Builder Interface
The module builder has 3 tabs:

**Tab 1: Lessons**
- Click "Add Lesson" to create a new lesson
- For each lesson, provide:
  - **Title**: Lesson name
  - **Type**: Choose Video, Text Content, or Document
  - **Duration**: Estimated time (e.g., "15 min")
  - **Content**: 
    - For videos: Paste YouTube/Vimeo URL
    - For text: Write your lesson content
    - For documents: Upload PDF or other files

**Tab 2: Quizzes**
- Click "Add Quiz" to create a quiz question
- For each quiz:
  - **Question**: Write your question
  - **Type**: Choose Multiple Choice, True/False, or Fill in the Blank
  - **Options**: For MCQ, fill in 4 options
  - **Correct Answer**: Specify the correct answer

**Tab 3: Assessments**
- Click "Add Assessment" for deeper evaluation
- For each assessment:
  - **Question**: Write the assessment prompt
  - **Type**: Choose Practical Application, Short Answer, or Matching Terms
  - **Expected Answer**: Provide grading criteria or sample answer

##### Save Your Module
- Review all tabs to ensure content is complete
- Click **"Save Module"** to add it to your course
- Repeat to add more modules

#### Edit or Delete Modules
- Click the **pencil icon** to edit a module
- Click the **trash icon** to delete a module
- Changes are saved when you complete the course creation

#### Upload Course Thumbnail
- Click the upload area to select an image
- Recommended size: 1280×720 pixels
- Formats: PNG, JPG, WEBP

#### Add Resources (Optional)
- Select resource type (PDFs or Videos)
- Upload files that supplement your course

Click **Next** to continue.

### Step 4: Set Pricing & Duration (Step 3 of 3)

#### Pricing
- Check "Free course" for a free course
- Or enter a price in Naira (₦)

#### Duration
- **Estimated Hours**: Total course duration
- **Hours/Week**: Expected weekly commitment
- **Duration**: Select schedule type (Self-paced or Scheduled)
- Use the calendar to set date range if scheduled

#### Publishing
- Check "Publish course immediately" to make it live
- Or leave unchecked to save as draft

#### Review Summary
- See total modules, lessons, quizzes, and assessments
- Default completion tokens: 10 (awarded when students finish)

Click **"Save Course"** or **"Update Course"** to finish!

## Tips for Success

### Module Organization
- Start with 3-5 modules for a complete course
- Each module should cover one major topic
- Order modules from basic to advanced

### Lesson Best Practices
- Keep video lessons under 20 minutes
- Mix content types (video + text + documents)
- Include 2-4 lessons per module

### Quiz Guidelines
- Add 1-2 quizzes per module
- Use quizzes to reinforce key concepts
- Mix question types for variety
- Provide clear, unambiguous questions

### Assessment Strategy
- Add 1 assessment per module for deeper evaluation
- Use practical assessments for hands-on skills
- Use short answer for conceptual understanding
- Provide detailed grading criteria

### Content Quality
- Use high-quality thumbnails
- Write clear, concise descriptions
- Proofread all content before publishing
- Test video links before saving

## Editing Existing Courses

1. Go to "My Courses" in your dashboard
2. Click the **Edit** button on any course
3. All existing modules, lessons, quizzes, and assessments will load
4. Make your changes
5. Click **"Update Course"** to save

## Student Experience

When students enroll in your course, they will:
1. See all modules in order
2. Complete lessons one by one
3. Take quizzes to test knowledge
4. Submit assessments for evaluation
5. Earn tokens for completing modules and quizzes
6. Receive a certificate upon course completion

## Token Rewards

Your students earn tokens automatically:
- **10 tokens** per module completed
- **5 tokens** per quiz passed
- **15 tokens** per assessment passed
- **10 tokens** for completing the entire course

These tokens can be used for premium features or converted to NFT certificates!
