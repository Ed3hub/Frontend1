# Educators Course Upload Guide

## Overview
The enhanced Upload Courses page now provides comprehensive module, lesson, quiz, and assessment management for educators.

## Features Added

### 1. Module Management
- Create, edit, and delete course modules
- Each module includes:
  - Title and description
  - Multiple lessons
  - Quizzes for knowledge checks
  - Assessments for deeper evaluation

### 2. Lesson Upload
Each lesson can be:
- **Video**: Upload video files or provide video URLs with duration
- **Text Content**: Write text-based lessons directly
- **Document**: Attach PDF or other document files

### 3. Quiz Creation
Three quiz types supported:
- **Multiple Choice**: 4 options with correct answer
- **True/False**: Binary choice questions
- **Fill in the Blank**: Text input questions

### 4. Assessment Creation
Three assessment types:
- **Practical Application**: Real-world scenario questions
- **Short Answer**: Open-ended questions requiring detailed responses
- **Matching Terms**: Match concepts with definitions

## How to Use

### Step 1: Course Details
- Enter course title, learning outcomes, ideal audience
- Set skill level and tools used

### Step 2: Course Structure
- Add modules using the "Add Module" button
- For each module:
  1. Switch between Lessons, Quizzes, and Assessments tabs
  2. Add content for each type
  3. Save the module
- Upload course thumbnail and resources
- View module summary showing total lessons, quizzes, and assessments

### Step 3: Pricing & Duration
- Set pricing (free or paid)
- Configure duration and schedule
- Publish immediately or save as draft

## Module Structure Example
```
Module 1: Introduction to DAOs
├── Lessons (3)
│   ├── Video: What is a DAO? (15 min)
│   ├── Text: DAO Governance Models
│   └── Document: DAO Case Studies.pdf
├── Quizzes (2)
│   ├── MCQ: Primary purpose of DAOs
│   └── True/False: Token-based voting
└── Assessments (1)
    └── Practical: Design a DAO governance flow
```

## Tips
- Add lessons first, then quizzes to test knowledge, then assessments for deeper evaluation
- Use descriptive titles for easy navigation
- Include duration estimates for better student planning
- Mix content types to keep learners engaged
